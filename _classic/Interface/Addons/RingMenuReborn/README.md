# RingMenu Reborn

A radial action wheel addon for WoW Classic Era that appears at your cursor position. Inspired by the original RingMenu addon that lost support in 2019.

## Features

- **Radial Action Wheels** - Circular menu that spawns at your cursor
- **Multiple Wheels** - Create separate wheels with independent keybinds and settings
- **Quick Cast Mode** - Hold keybind to show, release to activate the hovered action
- **Customizable Layout** - 1-24 buttons per wheel, adjustable radius and rotation
- **Visual Options** - Background tint with opacity control
- **Masque Support** - Compatible with Masque button skinning
- **Combat Safe** - Uses secure action templates for in-combat use

## Installation

1. Download and extract the `RingMenuReborn` folder
2. Place it in your `Interface/AddOns` directory
3. Restart WoW or `/reload`

## Usage

### Opening Settings
- `/rmr` or `/radialwheel` - Opens the configuration panel
- Or navigate to **Game Menu** > **Options** > **AddOns** > **RingMenu Reborn**

### Configuration Options
| Setting | Description |
|---------|-------------|
| Key Binding | Keybind to show/hide the wheel |
| Label | Optional name for the wheel |
| Starting Slot | First action bar slot (default: 13) |
| Button Count | Number of buttons (1-24) |
| Quick Cast | Hold to show, release to activate |
| Dismiss on Use | Auto-hide after clicking a button |
| Background Tint | Wheel background color and opacity |
| Wheel Radius | Distance of buttons from center |
| Rotation Offset | Rotate the wheel orientation |

### Adding Actions
1. Press your keybind to show the wheel
2. Drag spells, items, or macros onto the buttons
3. Click to activate or use Quick Cast mode

### Multiple Wheels
Use the **+** and **-** buttons in settings to create/delete wheels. Each wheel has its own keybind and configuration.

## Action Bar Slots

The addon uses standard action bar slots. Default starts at slot 13. If you encounter conflicts with stance bars or other addons, adjust the Starting Slot setting.

## Requirements

- WoW Classic Era (Interface 11506 / 1.15.x)
