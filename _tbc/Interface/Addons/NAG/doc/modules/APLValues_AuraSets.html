Module APLValues_AuraSets

" APL Values and Actions for Aura Sets functionality in the NAG addon.

This module provides functions for checking and managing aura sets, including item stat procs, trinket procs, stat buff cooldowns, and related APL values and actions. Maps to aplvaluesaura_sets.go in the Go simulation.

NAG:AllTrinketStatProcsActive (statType1, statType2, statType3, minIcdSeconds)
All Item Proc Buffs Active True if all proc trinkets (not use trinkets) that buff the specified stat type(s) are currently active, otherwise False. For stacking proc buffs, this condition also checks that the buff has been stacked to its maximum possible strength.

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)
  • minIcdSeconds ? number Minimum ICD threshold in seconds (defaults to 0)

Returns:

    boolean if all matching proc trinkets have active procs, or if none are registered

Usage:

    local allActive = NAG:AllTrinketStatProcsActive(Types:GetType("Stat").HASTE)
NAG:AnyStatBuffCooldownsActive (statType1, statType2, statType3)
Any Stat Buff Cooldowns Active True if any use trinkets (Major Cooldowns) that buff the specified stat type(s) are currently active, otherwise False. For stacking buffs, this condition also checks that the buff has been stacked to its maximum possible strength after the cooldown is activated. Both manually casted cooldowns as well as cooldowns controlled by "Cast All Stat Buff Cooldowns" and "Autocast Other Cooldowns" actions are checked.

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)

Returns:

    boolean if any matching use trinket cooldowns are active

Usage:

    local anyActive = NAG:AnyStatBuffCooldownsActive(Types:GetType("Stat").HASTE)
NAG:AnyStatBuffCooldownsMinDuration (statType1, statType2, statType3)
Gets the minimum remaining duration among active stat buff cooldowns (use trinkets) for the requested stat types. Returns 999 when no matching cooldown buffs are currently active to mirror simulator behavior.

Parameters:

  • statType1 ? any First stat type to check (supports strings, numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings, numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings, numbers, or nil)

Returns:

    number remaining duration in seconds, or 999 if none are active

Usage:

    local minDuration = NAG:AnyStatBuffCooldownsMinDuration(Types:GetType("Stat").HASTE)
NAG:AnyTrinketStatProcsActive (statType1, statType2, statType3, minIcdSeconds)
Any Item Proc Buff Active True if any proc trinkets (not use trinkets) that buff the specified stat type(s) are currently active, otherwise False. For stacking proc buffs, this condition also checks that the buff has been stacked to its maximum possible strength.

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)
  • minIcdSeconds ? number Minimum ICD threshold in seconds (defaults to 0)

Returns:

    boolean if any matching proc trinket has an active proc

Usage:

    local anyActive = NAG:AnyTrinketStatProcsActive(Types:GetType("Stat").HASTE)
NAG:AnyTrinketStatProcsAvailable (statType1, statType2, statType3, minIcdSeconds)
Checks if any proc trinket that matches the requested stat types is currently available to trigger. Availability is defined as the proc aura being inactive while the internal cooldown is ready.

Parameters:

  • statType1 ? any First stat type to check (supports strings, numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings, numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings, numbers, or nil)
  • minIcdSeconds ? number Minimum ICD threshold in seconds (optional)

Returns:

    boolean if any matching proc trinket can proc immediately

Usage:

    if NAG:AnyTrinketStatProcsAvailable(Types:GetType("Stat").HASTE) then ... end
NAG:NumEquippedStatProcTrinkets (statType1, statType2, statType3, minIcdSeconds)
Num Equipped Stat Proc Effects Number of equipped passive item/enchant effects that buff the specified stat type(s) when they proc.

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)
  • minIcdSeconds ? number Minimum ICD threshold in seconds (defaults to 0)

Returns:

    number of matching equipped trinkets

Usage:

    local count = NAG:NumEquippedStatProcTrinkets(Types:GetType("Stat").HASTE)
NAG:NumStatBuffCooldowns (statType1, statType2, statType3)
Num Stat Buff Cooldowns Number of registered Major Cooldowns that buff the specified stat type(s) when they are cast. Both manually casted cooldowns as well as cooldowns controlled by "Cast All Stat Buff Cooldowns" and "Autocast Other Cooldowns" actions are included in the total count returned by this value. NOTE: Currently only counts equipped OnUse trinkets (major cooldowns not yet implemented)

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)

Returns:

    number of equipped OnUse trinkets that provide matching stat buffs

Usage:

    local numBuffs = NAG:NumStatBuffCooldowns(Types:GetType("Stat").HASTE)
NAG:TrinketProcsMaxRemainingICD (statType1, statType2, statType3, minIcdSeconds)
Item Procs Max Remaining ICD Longest remaining ICD on any inactive proc trinkets (not use trinkets) that buff the specified stat type(s), or 0 if all are currently active.

Parameters:

  • statType1 ? any First stat type to match (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to match (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to match (supports strings like "haste", numbers, or nil)
  • minIcdSeconds ? number Minimum ICD in seconds to consider (optional)

Returns:

    number maximum remaining ICD time in seconds, or 0 if all matching proc trinkets are active

Usage:

    local maxIcd = NAG:TrinketProcsMaxRemainingICD(Types:GetType("Stat").HASTE)

Item Procs Min Remaining Time Shortest remaining duration on any active proc trinkets (not use trinkets) that buff the specified stat type(s), or infinity if none are currently active.

Parameters:

  • statType1 ? any First stat type to check (supports strings like "str", numbers, or nil)
  • statType2 ? any Second stat type to check (supports strings like "crit", numbers, or nil)
  • statType3 ? any Third stat type to check (supports strings like "haste", numbers, or nil)
  • minIcdSeconds ? number Minimum ICD in seconds to consider (optional)

Returns:

    number remaining time among matching proc trinkets, or 999 if none are currently active

Usage:

    local minTime = NAG:TrinketProcsMinRemainingTime(Types:GetType("Stat").HASTE)
