Module APLAction_Casting

" Core APL logic, utility, funnel, and casting handlers for the NAG addon

This module provides utility functions for spell casting, last cast tracking, and aura cancellation.

Automatically casts other cooldowns based on settings and availability. Maps to APLActionAutocastOtherCooldowns in Go. When enableRacials is true, suggests throughput racials (Blood Fury, Berserking, etc.) when ready; position comes from class specSpellLocations (LEFT) so APLs that call this get racials without listing them.

  • enableGloveSlot boolean |nil Whether to use the glove slot item (default: true)
  • enableBeltSlot boolean |nil Whether to use the belt slot item (default: false)

    boolean returns false (adds items to secondary spells list).

NAG:CastSpell (id, tolerance, position, hideWhileCasting, overlayText)
Attempts to cast a spell, trinket, tinker, or item. This is the primary casting API and maps to APLActionCastSpell in Go.

Parameters:

  • id number |number[] The ID of the spell, trinket, tinker, or item, or an ordered list of spell IDs to try.
  • tolerance number |nil Optional casting tolerance in seconds (default: 0)
  • position string |nil Optional position override (use NAG.SPELL_POSITIONS constants)
  • hideWhileCasting boolean |nil Optional: if true, returns false if the spell is currently being cast or channeled.
  • overlayText string |nil Optional: overlay text to display with the spell cast.

Returns:

    boolean if the cast was successful, false otherwise.

Usage:

  • NAG:CastSpell(12345) -- Normal cast
  • NAG:CastSpell(12345, 0.5) -- With tolerance
  • NAG:CastSpell(12345, nil, ns.SPELL_POSITIONS.RIGHT) -- With position
  • NAG:CastSpell(12345, 0.5, ns.SPELL_POSITIONS.RIGHT) -- With both
  • NAG:CastSpell(12345, nil, nil, nil, "TEXT") -- With overlay text
