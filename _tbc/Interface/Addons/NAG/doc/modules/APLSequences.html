NAG:Sequence (name, forceMain, ...)
Initializes a progressive sequence of actions (spell IDs or functions) to be executed in order. A Sequence progresses after each spell cast, preserving state between rotation cycles. Does NOT require all spells to be ready before starting (unlike StrictSequence).

Parameters:

  • name string The name of the sequence.
  • forceMain boolean |nil Optional flag to force first spell to main frame (default: true).
  • ... any Variable arguments: spell IDs or functions to perform in sequence.

Returns:

    boolean if the sequence is active and should block main rotation, false otherwise.

Usage:

  • NAG:Sequence("sequenceName", nil, spell1, spell2, spell3)
  • NAG:Sequence("sequenceName", true, spell1, spell2, spell3) -- Force first spell to main frame

NAG:StrictSequence (name, forceMain, ...)
Initializes a strict sequence of actions (spell IDs or wait commands) to be executed in order. A StrictSequence only activates when ALL listed actions are ready and castable at once.

Parameters:

  • name string The name of the sequence.
  • forceMain boolean |nil Optional flag to force first spell to main frame (default: true).
  • ... any Variable arguments: spell IDs (number) or wait commands (string) to perform in sequence.

Returns:

    boolean if the sequence is active and should block main rotation, false otherwise.

See also:

Usage:

  • NAG:StrictSequence("sequenceName", 12345) -- forceMain optional: one action => defaults to true
  • NAG:StrictSequence("sequenceName", nil, 12345, "NAG:Wait(2)", 67890)
  • NAG:StrictSequence("sequenceName", true, 12345, 67890) -- Force first spell to main frame
