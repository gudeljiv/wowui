Module APLDRW

" Dancing Rune Weapon (DRW) tracking system for Death Knight

This module provides tracking for DRW summons, debuffs, and expiration for the NAG addon.

NAG:NumberTargetsWithDRWDebuff (spellId, range)
Gets the number of targets in range that have a specific debuff applied by DRW.

Parameters:

  • spellId number The ID of the debuff to check for.
  • range number |nil Optional range to use for counting targets (default: 15).

Returns:

    number number of targets with the DRW-applied debuff.

Usage:

  • NAG:NumberTargetsWithDRWDebuff(55095) -- Count targets with DRW's Frost Fever
  • NAG:NumberTargetsWithDRWDebuff(55078, 10) -- Count targets within 10 yards with DRW's Blood Plague
NAG:TargetDRWDebuffRemainingTime (spellId)
Gets the remaining time for a DRW-applied debuff on the current target.

Parameters:

  • spellId number The ID of the debuff to check for.

Returns:

    number remaining time in seconds, or 0 if not found.

Usage:

    NAG:TargetDRWDebuffRemainingTime(55095) -- Get remaining time for DRW's Frost Fever
NAG:TargetHasDRWDebuff (spellId)

Parameters:

  • spellId number The ID of the debuff to check for.

Returns:

    boolean if the target has the DRW-applied debuff, false otherwise.

Usage:

    NAG:TargetHasDRWDebuff(55095) -- Check if target has DRW's Frost Fever
