Module APLValues_Boss

" Boss-related APL value functions for the NAG addon

NAG:BossCastTimeRemaining ()
Gets the remaining cast time for the boss's current spell.

Returns:

    number cast time in seconds, or 0 if not casting or no boss target

Usage:

    NAG:BossCastTimeRemaining() <= 2.5 -- Check if cast will finish within 2.5 seconds
NAG:BossChannelTimeRemaining ()
Gets the remaining channel time for the boss's current spell.

Returns:

    number channel time in seconds, or 0 if not channeling or no boss target

Usage:

    NAG:BossChannelTimeRemaining() <= 3.0 -- Check if channel will finish within 3 seconds
NAG:BossCurrentTarget (targetUnit)
Checks if the specified unit's current target is the player. Mirrors the simulator's BossCurrentTarget value which evaluates to true when the boss is actively targeting the player (i.e., the player is tanking).

Parameters:

  • targetUnit ? string Unit token for the boss (defaults to the first active boss unit or "target")

Returns:

    boolean if the boss is targeting the player, false otherwise
NAG:BossSpellIsCasting (targetUnit, spellId)
Checks if the boss is currently casting or channeling a specific spell.

Parameters:

  • targetUnit ? string The target unit to check (defaults to the first active boss unit or "target")
  • spellId number The spell ID to check for (returns false if nil or invalid)

Returns:

    boolean if the boss is casting or channeling the specified spell, false otherwise

Usage:

    NAG:BossSpellIsCasting("target", 12345) -- Check if boss is casting or channeling spell ID 12345
NAG:BossSpellTimeToReady (targetUnit, spellId)
Gets the time until a boss spell is ready (off cooldown).

Parameters:

  • targetUnit ? string The target unit to check (defaults to "target")
  • spellId number The spell ID to check (returns 0 if nil or invalid)

Returns:

    number in seconds until the spell is ready, or 0 if already ready or invalid spellId

Usage:

    NAG:BossSpellTimeToReady("target", 12345) <= 10 -- Check if spell will be ready within 10 seconds
