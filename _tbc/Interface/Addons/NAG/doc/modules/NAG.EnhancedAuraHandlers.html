NAG:CountRaidMembersWithAura (spellId, filter)
Counts unique group/raid members who have a player-cast aura active.

Parameters:

  • spellId number |string The spell ID or name of the aura to count
  • filter string |nil Optional filter (defaults to "HELPFUL|PLAYER" for player-cast buffs)

Returns:

    number The count of unique party/raid members with the aura (1 if solo)

Usage:

  • local count = NAG:CountRaidMembersWithAura(12345) -- Count player-cast buffs
  • local count = NAG:CountRaidMembersWithAura(12345, "HARMFUL|PLAYER") -- Count player-cast debuffs

NAG:UnitAura (unit, spellId, filter, exactIdMatch)
Core function that handles all aura detection logic with integrated smart detection.

Parameters:

  • unit string The unit to check
  • spellId number |string The spell ID or name to check
  • filter string |nil Optional filter (e.g., "HELPFUL", "HARMFUL", "HARMFUL|PLAYER")
  • exactIdMatch boolean |nil If true, forces exact spell ID matching and skips name-based matching

Returns:

    AuraData |boolean AuraData table if found, false if not found

Usage:

  • local aura = NAG:UnitAura("player", 12345, "HELPFUL")
  • if aura then local name, icon = aura.name, aura.icon end
