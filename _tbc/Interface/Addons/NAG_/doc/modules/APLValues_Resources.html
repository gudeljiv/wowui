NAG:CurrentManaPercentBase (sourceUnit)
Get the current mana percentage based on BASE mana (not total mana pool) This is useful for powershifting checks where you want to know mana percentage relative to your base mana pool, ignoring temporary mana pool increases from buffs.

Parameters:

  • sourceUnit ? string The unit to check mana percentage for (defaults to "player")

Returns:

    number current mana percentage of base mana (0-1 decimal range), or current/total if base cannot be calculated

Usage:

    NAG:CurrentManaPercentBase() > 0.35
