NAG:BrewmasterMonkCurrentStaggerPercent ()
Returns the current stagger percentage for a Brewmaster Monk. Calculates the percentage of maximum health currently being staggered by checking Light, Moderate and Heavy stagger debuffs.

Returns:

    number current stagger amount as a percentage of max health (0-100)

Usage:

    local staggerPct = NAG:BrewmasterMonkCurrentStaggerPercent()

NAG:DruidCurrentEclipsePhase ()
Determine the current Eclipse phase for a Balance Druid. Returns "SolarPhase", "LunarPhase", or "NeutralPhase" if neither is active. Uses GetEclipseDirection API when available, falls back to aura checking.

Returns:

    string current Eclipse phase ("SolarPhase", "LunarPhase", or "NeutralPhase").

Usage:

    NAG:DruidCurrentEclipsePhase() == "SolarPhase"
NAG:MageCurrentCombustionDotEstimate ()
Returns the estimated DoT damage for a Mage's Combustion. Uses actual tracked Ignite values to calculate Combustion damage, accounting for Glyph of Combustion, haste scaling, and crit chance. Only considers Ignite as the source DoT.

Returns:

    number estimated total damage from Combustion based on current Ignite state.

Usage:

    NAG:MageCurrentCombustionDotEstimate()
