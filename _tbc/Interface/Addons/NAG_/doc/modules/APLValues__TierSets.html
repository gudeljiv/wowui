Class NAG

NAG:TierSetEquipped (tier, count)
Checks if a specific tier set is equipped. Determines if the player has at least the specified number of pieces for a given tier set.

Parameters:

  • tier string The tier set identifier to check (e.g., "Druid_T30")
  • count number The minimum number of pieces required (1-6)

Returns:

    boolean if the player has at least 'count' pieces of the specified tier set, false otherwise

See also:

Usage:

    if NAG:TierSetEquipped("Druid_T30", 4) then
    print("4-piece Druid T30 bonus active!")
elseif NAG:TierSetEquipped("Druid_T30", 2) then
    print("2-piece Druid T30 bonus active!")
end
