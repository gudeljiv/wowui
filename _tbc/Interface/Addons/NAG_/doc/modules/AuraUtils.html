Class NAG

NAG:DotIsActiveByPlayer (spellID, unit, includePet)
Returns true if a debuff (by spellID) is active on unit and cast by player (or optionally pet)

Parameters:

  • spellID number The debuff spell ID to check
  • unit string |nil Unit to scan (defaults to "target")
  • includePet boolean |nil If true, counts pet/guardian as caster as well

Returns:

    boolean
