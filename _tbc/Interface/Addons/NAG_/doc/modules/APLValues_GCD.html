Module APLValues_GCD

" GCD (Global Cooldown) related APL value functions for the NAG addon

This module provides utility functions for GCD timing, cooldown management, and spell readiness.

Class NAG

NAG:GCDDebugSnapshot ()
Returns a debug snapshot of the raw GCD probe cooldown and computed remaining. Useful for verifying "duration vs remaining" behavior on Classic/TBC clients.

Returns:

    table
NAG:GCDTimeToReady ()
Returns the time until the global cooldown (GCD) is ready.

Returns:

    number time until the GCD is ready in seconds (0 if ready).

Usage:

    local timeLeft = NAG:GCDTimeToReady()
if timeLeft <= 0.5 then
    -- GCD will be ready soon
end
NAG:GCDTimeValue ()
Returns the last observed Global Cooldown (GCD) duration value in seconds. When GCD is ready, GetSpellCooldown returns duration 0, so we cache the last non-zero value and return it. Uses unified API and Classic/Retail IDs.

Returns:

    number last non-zero GCD duration in seconds (defaults to 1.5 if never observed)

Usage:

    local gcdDuration = NAG:GCDTimeValue()
 -- Use for timing calculations when GCD is not currently active
