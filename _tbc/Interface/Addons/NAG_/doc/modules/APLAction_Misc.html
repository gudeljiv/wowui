NAG:ActivateAllStatBuffProcAuras (swapSet, statType1, statType2, statType3)
Activates all stat buff proc auras (not yet implemented). Maps to APLActionActivateAllStatBuffProcAuras in Go.

Parameters:

  • swapSet string |number The item swap set to use (ItemSwapSet.Main, ItemSwapSet.Swap1, etc.)
  • statType1 number First stat type to match (-1 for any)
  • statType2 number Second stat type to match (-1 for any)
  • statType3 number Third stat type to match (-1 for any)

Returns:

    boolean returns false (not implemented)
NAG:ActivateAura (auraId)
Activates an aura (alias to Cast for APL compatibility). Maps to APLActionActivateAura in Go.

Parameters:

  • auraId number The ID of the aura to activate

Returns:

    boolean of the cast operation
NAG:ActivateAuraWithStacks (auraId, numStacks)
Activates an aura with a specific number of stacks. Maps to APLActionActivateAuraWithStacks in Go.

Parameters:

  • auraId number The ID of the aura to activate
  • numStacks number The number of stacks to apply

Returns:

    boolean returns true

Usage:

    NAG:ActivateAuraWithStacks(12345, 3)

Cancels an aura if it is active and shows an overlay for cancellation. Maps to APLActionCancelAura in Go.
